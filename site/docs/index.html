Getting Started

Installation

Install the CLI: npm install -g @blackroad/cli

Quick Start

Run br brand init to scaffold your brand config, then br brand site to generate your site.

Templates

Use br brand list to see all 15 available templates.

Deploy

Run br brand deploy --project my-site --dir ./site to publish to Cloudflare Pages.
